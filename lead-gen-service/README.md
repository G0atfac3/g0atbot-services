# Automated Lead Generation Bot

Scans for leads and initiates outreach.

## Run Campaign
```bash
python3 lead_gen.py run "marketing agency"
```

## List Leads
```bash
python3 lead_gen.py list
```

## What's It Do
1. Searches for companies in target industries
2. Enriches with contact data
3. Scores based on fit (industry, size, etc.)
4. Sends personalized outreach

## Target Industries
- Marketing agencies
- SEO agencies
- Web design
- SaaS startups
- Ecommerce
- Crypto companies

## Files
- `leads.csv` - Lead database
- `outreach.json` - Outreach history
